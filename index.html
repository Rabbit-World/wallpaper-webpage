Lively

An Animated Windows Wallpaper system.

Get Started

An opensourced animated wallpaper system that allows you to create animated wallpaper from your favourite piece of media by just drag and drop.

It include support for, but not limited to :
  • Multiple Monitors
  • Automagic audio handling for video wallpapers
  • Goes to Sleep (~0% CPU) upon launching a fullscreen application
  • Built-in Lively supported wallpaper creator for sharing your creation with the world
  • Ability to combine other Windows desktop customization tools like Rainmeter - i.e Infinite creativity
